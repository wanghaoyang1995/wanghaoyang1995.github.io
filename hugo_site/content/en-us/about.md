### This Site

This is a static personal blog hosted on [*Github Pages*](https://pages.github.com/).

It’s built with the [*Hugo*](https://gohugo.io/) static site generator, styled with the [Layui](https://layui.dev/) UI library, and loosely inspired by the [*Mainroad*](https://github.com/Vimux/Mainroad) theme. Comments are powered by the [*Giscus*](https://github.com/giscus/giscus) integration.

Since private repositories on GitHub Pages require a paid plan, this site is reluctantly open-sourced. You can find the repository [here](https://github.com/wanghaoyang1995/wanghaoyang1995.github.io).
